# 📖 สมุดเช็คชื่อ เซนต์หลุยส์ 2026

เว็บแอปเช็คชื่อนักเรียน สำหรับโรงเรียน/ชั้นเรียนคำสอน **เซนต์หลุยส์** — เช็คชื่อรายวัน, สแกน QR หน้าห้อง, และดูว่า **ใครขาดหน้าไหนบ้าง** ได้ในหน้าเดียว ทำงานได้ทั้งบนคอมพิวเตอร์และมือถือ ไม่ต้องติดตั้งอะไร ไม่ต้องมีเซิร์ฟเวอร์

> ดัดแปลงจากไฟล์ Notepad (`timeattendanceSTLouis2026.txt`) ให้เป็นเว็บแอปที่เปิดใช้งานจริงบนอินเทอร์เน็ตได้ฟรีผ่าน GitHub Pages

---

## ✨ ความสามารถ

| แท็บ | ทำอะไรได้ |
|------|-----------|
| **เช็คชื่อวันนี้** | เลือกวันที่ แล้วกดสถานะ มา / สาย / ขาด ให้นักเรียนแต่ละคน พร้อมบันทึกเวลา ปุ่ม "มาทั้งหมด" เช็คทุกคนรวดเดียว |
| **สแกน QR หน้าห้อง** | เปิดกล้อง ให้นักเรียนสแกน QR ของตัวเอง ระบบเช็คชื่อ + บันทึกเวลาอัตโนมัติ พร้อมเสียงและไฟกระพริบยืนยัน |
| **จัดการรายชื่อ** | เพิ่ม / ลบ รายชื่อนักเรียน |
| **QR นักเรียน** | สร้าง QR โค้ดประจำตัวนักเรียนทุกคน พิมพ์แจก หรือให้ถ่ายเก็บไว้ |
| **ตารางเรียน** | กำหนดวันเรียนและหัวข้อในแต่ละครั้ง (ค่าเริ่มต้นเป็นวันอาทิตย์) |
| **🚩 ใครขาดบ้าง** | **ดูว่านักเรียนแต่ละคนขาดวันไหนไปบ้าง** เทียบวันเรียนที่ผ่านมากับการเช็คชื่อ เรียงคนที่ขาดมากสุดขึ้นก่อน กรองเฉพาะคนที่มีขาดได้ |
| **สรุปสถิติ** | สรุป มา/สาย/ขาด และ % การเข้าเรียนของทุกคน |

### 🚩 "ใครขาดหน้าไหนบ้าง" ทำงานอย่างไร
- นับ **"วันเรียน"** จากแท็บ *ตารางเรียน* เฉพาะวันที่ถึงกำหนดแล้ว (ไม่นับวันในอนาคต)
- นักเรียนที่ถูกทำเครื่องหมาย **"ขาด"** หรือ **ยังไม่ถูกเช็คชื่อ** ในวันนั้น จะถือว่า **ขาด**
- สถานะ **"สาย"** นับว่า *มาเรียน* (ไม่ถือเป็นขาด)
- แสดงเป็นชิปวันที่ที่ขาด และส่งออกเป็น Excel ได้

### ส่งออกข้อมูล
- **Excel (.xlsx)** — สรุปสถิติ, รายละเอียดรายวัน, บันทึกเวลาเข้าเรียน, และรายชื่อคนขาด
- **PDF / พิมพ์** — พิมพ์รายงานหรือบัตร QR ได้จากเบราว์เซอร์

## 🚀 วิธีใช้งาน

1. เปิดเว็บ (ดูลิงก์ด้านล่าง) หรือดับเบิลคลิกไฟล์ `index.html`
2. ไปที่แท็บ **จัดการรายชื่อ** เพิ่มรายชื่อนักเรียน
3. ไปที่แท็บ **ตารางเรียน** กำหนดวันเรียนแต่ละครั้ง
4. แต่ละวันเรียน ใช้ **เช็คชื่อวันนี้** หรือ **สแกน QR หน้าห้อง**
5. อยากรู้ว่าใครขาดไปกี่ครั้ง วันไหนบ้าง → เปิดแท็บ **🚩 ใครขาดบ้าง**

### 📷 เรื่องกล้อง (สแกน QR)
กล้องต้องใช้งานผ่าน **HTTPS** — ลิงก์ GitHub Pages เป็น HTTPS อยู่แล้วจึงใช้กล้องได้เลย (แค่กด "อนุญาต" ตอนเบราว์เซอร์ถาม) หากเปิดไฟล์แบบ `file://` บนเครื่อง เบราว์เซอร์บางตัวอาจบล็อกกล้อง

## 💾 การเก็บข้อมูล

ข้อมูลทั้งหมด (รายชื่อ, การเช็คชื่อ, ตารางเรียน) ถูกเก็บไว้ใน **`localStorage` ของเบราว์เซอร์เครื่องนั้น ๆ** เท่านั้น
- ไม่มีการส่งข้อมูลขึ้นเซิร์ฟเวอร์ ปลอดภัยและเป็นส่วนตัว
- **ข้อควรระวัง:** ข้อมูลผูกกับเบราว์เซอร์+เครื่องที่ใช้ ถ้าเปลี่ยนเครื่อง/ล้างข้อมูลเบราว์เซอร์ ข้อมูลจะหาย → แนะนำ **ส่งออก Excel เก็บสำรองเป็นระยะ**
- ควรใช้เครื่อง/เบราว์เซอร์ตัวเดิมเป็นหลักในการเช็คชื่อ

## 🌐 นำขึ้นเว็บฟรีด้วย GitHub Pages

1. สร้าง repository บน GitHub แล้ว push โค้ดชุดนี้ขึ้นไป (ไฟล์ `index.html` ต้องอยู่ที่ root)
2. ไปที่ **Settings → Pages**
3. ที่ **Source** เลือก **Deploy from a branch** → branch `main` → โฟลเดอร์ `/ (root)` → **Save**
4. รอสักครู่ จะได้ URL ฟรีรูปแบบ:
   ```
   https://<username>.github.io/<repo-name>/
   ```
